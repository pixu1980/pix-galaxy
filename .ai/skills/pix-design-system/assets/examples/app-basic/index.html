__BRAND_NAME__

App-mode design system baseline with proportional foundations, layout primitives, helper attributes, and component hooks left as placeholders.

Typography ratio

Heading and body sizes derive from a base font size and a ratio token through `pow()`.

Spacing ratio

Layout gaps derive from a base spacing value, a spacing ratio, and an optional density multiplier.

Component hooks

The components layer documents selectors such as `data-component="button"` without shipping their implementations.
