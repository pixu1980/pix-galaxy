# Code of Conduct

This project is part of the pix-galaxy suite and is committed to a respectful, inclusive, and technically constructive community.

## Expected Behavior

- Be respectful in disagreement and precise in technical discussion.
- Assume good intent, but challenge ideas with evidence.
- Keep feedback actionable and focused on code, docs, or process.
- Help make the project accessible to contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, intimidation, dogpiling, or personal attacks.
- Discrimination based on identity, background, disability, religion, nationality, age, or orientation.
- Sexualized language or unwanted sexual attention.
- Publishing private information without explicit permission.
- Repeated bad-faith review behavior, trolling, or deliberate discussion derailment.

## Maintainer Responsibilities

Maintainers may edit, remove, or reject comments, commits, issues, discussions, and pull requests that violate this code of conduct. They may also warn, suspend, or ban participants when needed to protect the project and its contributors.

## Scope

This code of conduct applies in the repository, issue tracker, pull requests, release discussions, and any project-managed communication channel.

## Reporting

Report conduct issues privately to the maintainers through GitHub at https://github.com/pixu1980. Include links, screenshots, or timestamps when possible so the report can be assessed quickly.

## Enforcement

Maintainers will review reports promptly, keep sensitive details private where possible, and apply consequences proportionate to the behavior. Repeated or severe violations may result in a permanent ban from project spaces.
