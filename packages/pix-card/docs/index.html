<pix-card>

Accessible card Web Component. Zero runtime dependencies.

Installation

pnpm add @pix-galaxy/pix-card

Usage

import "@pix-galaxy/pix-card";
import "@pix-galaxy/pix-card/css";
<pix-card>Card content</pix-card>

Examples

Default Card Header Default card body content Footer Outlined Card Outlined card body Elevated Card Elevated card body Link Card This card is a link — try keyboard navigation

Keyboard Interaction

When href is set, the card renders as an <a> element and supports standard link keyboard navigation (Enter to activate, Tab to focus).

Accessibility

  • When href is set, the card uses a native <a> element
  • Focus outline preserved via :focus-visible
  • Supports forced colors (high contrast)
  • Semantic slot structure for header, body, and footer
