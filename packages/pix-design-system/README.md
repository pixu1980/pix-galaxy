# @pix-galaxy/pix-design-system

Native CSS design system package for the pix-galaxy monorepo.

## Usage

```js
import '@pix-galaxy/pix-design-system/css';
```

## Source of Truth

The package bundles the shared pix design system CSS from the local skill source under `.ai/skills/pix-design-system/assets/design-system/shared/styles/`.
