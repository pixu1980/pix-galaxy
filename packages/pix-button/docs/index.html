<pix-button>

Accessible button Web Component. Zero runtime dependencies.

Installation

pnpm add @pix-galaxy/pix-button

Usage

import "@pix-galaxy/pix-button";
import "@pix-galaxy/pix-button/css";
<pix-button>Click me</pix-button>

Examples

Primary Secondary Ghost Disabled

Keyboard Interaction

The button is a native <button> element and supports all standard keyboard interactions including Enter and Space to activate.

Accessibility

  • Uses a native <button> element inside Shadow DOM
  • Supports keyboard activation (Enter, Space)
  • Supports forced colors (high contrast mode)
  • Supports reduced motion
  • Focus ring preserved via :focus-visible
